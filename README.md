#### A customized version of the [Olsen Framework](https://github.com/dklollol/Olsen-Framework-Arma-3) for use by [2. Panzer Division](http://2panzer.wixsite.com/home) ARMA mission makers as a mission template. 

#### Required dependencies:
* ARMA 3
* 2.Pzd Modset

#### Usage Guide:
* Extract the composition and mission folders to your ARMA 3 profile folder.
* Follow the mission making guide located [here](http://2panzer.forumpro.eu/t437-haas-s-mission-making-guide-for-the-olsen-framework) on the 2.PzD forum.
